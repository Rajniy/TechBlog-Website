# TechBlog - A Modern Blog Website with Authentication

## üìñ Project Description

TechBlog is a dynamic blog website featuring user authentication, article management, and an intuitive dashboard. Built with a robust Python Flask backend and a responsive vanilla JavaScript frontend, it offers a seamless experience for reading and managing tech-related content. The platform supports user registration, login, article browsing by categories, and theme switching (light/dark mode).

- **User Authentication**: Secure signup and login with JWT-based authentication
- **Article Management**: Create, edit, view, and manage blog articles
- **Category System**: Organize articles by topics like Web Development, AI, Graphic Design, etc.
- **Dashboard**: User dashboard for managing personal content and profile
- **Responsive Design**: Optimized for desktop, tablet, and mobile devices
- **Search Functionality**: Find articles by keywords or categories
- **Theme Toggle**: Switch between light and dark themes
- **Sample Articles**: Pre-seeded with demo content for immediate testing

- **A modern web browser** (Chrome, Firefox, Safari, Edge)

## üöÄ Installation and Setup

1. Clone the repository and navigate to the project directory.
2. Create a `.env` file with a generated SECRET_KEY.
3. Set up virtual environment: `python -m venv venv` and activate it.
4. Install dependencies: `pip install flask flask-cors python-dotenv`
5. Run backend: `cd backend && python app.py`
Frontend: Open `index.html` in browser.

## üéØ Usage

1. **Start the Backend**: Run `python backend/app.py`
2. **Open the Website**: Open `index.html` in your browser
3. **Sign Up**: Create a new account on the signup page
4. **Log In**: Use your credentials to access the platform
5. **Browse Articles**: Explore articles on the home page or by categories
6. **Dashboard**: Access your dashboard to manage content (requires login)
7. **Theme Toggle**: Switch between light and dark modes using the moon/sun icon
